# Case 02 — 시연 대본 (단가 검증 + Discord 이상치 알림)

## 1분 버전

**[10초] 페인**
> "최주임이 매일 거래명세서 100건 단가·수량을 수기 대조 — 1~2시간. 가끔 놓치면 손실로 직결됩니다."

**[5초] 실행**
```bash
uv run python -m cases.case02_excel_invoice_validation.scenario
```

**[30초] 결과**
- 콘솔: "이상치 5건 검출"
- Discord 채널 화면 전환 → 알림 도착 ("거래명세서 단가 이상치 알림")
- Quick Look으로 outliers.xlsx 열기 (단가 컬럼 강조)

**[15초] 임팩트**
> "사람이 100건을 다 보지 않아도 됩니다. 의심되는 것만 즉시 채널로. 단가 강결합 없이 다른 품목 마스터도 바로 적용 가능합니다."
