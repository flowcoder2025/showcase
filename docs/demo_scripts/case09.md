# Case 09 — 시연 대본 (AI 메일 초안)

## 1분 버전

**[10초] 페인**
> "이대리는 거래처 답신 한 통에 10분씩 톤 고민. 하루 50통이면 8시간."

**[5초] 실행**
```bash
DEMO_SAFE=1 uv run python -m cases.case09_ai_email_drafter.scenario
```
→ 안전 모드 권장 (캐시 응답 — 시연 안정)

**[30초] 결과**
- drafts.json 열기 → 3가지 톤 비교
  - 옵션 1: 신중·보수
  - 옵션 2: 친근·관계 강화
  - 옵션 3: 명확·간결
- "ChatGPT랑 뭐가 달라요?" → 사내 톤 + 거래 이력이 prompt에 자동 주입됨을 강조

**[15초] 임팩트**
> "단순한 글쓰기가 아니라 이 회사 거래 이력에 맞춘 답신입니다. 영업팀 신입도 부장 톤으로 씁니다."

## 3분 버전
1분 + 입력 메일 보여주기 + COMPANY_TONE 변수 보여주기 + 톤 바꿔서 1회 더 실행

## 5분 버전 (강의)
3분 + 모델 폴백 시연 (primary 차단 → fallback 자동) + 비용·지연 비교
